# WeatherRecruitmentApp

This is an application about showing weather data for 5 days from today.

## Technology stack

In this application i used below libraries:
- Kotlin
- Retrofit2
- Courutine
- Flow 
- Navigation Component
- DaggerHilt

Also application was made in MVVM architecture

## Installation

Clone this repository and import into **Android Studio**
```bash
git@github.com:P1entaszek/WeatherRecruitmentApp.git
```

## Configuration

Application doesn't required specific configuration

## Pre-requisites

- Android SDK 31

## Generating signed APK
From Android Studio:
1. ***Build*** menu
2. ***Generate Signed APK...***
3. Fill in the keystore information *(you only need to do this once manually and then let Android Studio remember it)*
